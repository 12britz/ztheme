## Long-term Knowledge

### Architecture

<!-- lore:019ed6b8-1316-744e-b8a3-967a1cf8cba0 -->
* **Category field on Theme for grouped CLI selection**: Category field on Theme for grouped CLI selection: Optional \`category: string\` organizes themes into groups ('popular', 'classic'). CLI groups themes by category with dim dashed section headers (\`── Popular ─────────────────────\`). Each option is prefixed with a \`◆\` diamond bullet colored in the theme's foreground, followed by the theme name (colored in its foreground) and author name dimmed after \`·\`. Cancel option indented with leading space. Per-option Separator lines removed to reduce clutter. After selection, compact palette preview shown (title bar in theme colors + 16-color swatch row). Improves discoverability and readability over flat list.

### Pattern

<!-- lore:019ed6ce-a464-759d-8799-225dc3f0ff1f -->
* **ASCII banner with per-character horizontal gradient on CLI launch**: ztheme CLI shows a block-letter 'ZTHEME' ASCII art banner on launch, using a per-character horizontal gradient (violet #a78bfa to teal #2dd4bf), matching zgit's banner style (src/banner.js). Displayed via showBanner() after console.clear() and before theme selection, replacing an older box header. Defined in src/banner.ts and called from cli.ts/index.ts. Chosen over a static banner for a distinctive identity consistent with the project's bold design aesthetic.
